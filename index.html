Grant Thornton AWS Solutions Group

The Grant Thornton AWS Solution Group accelerates end-to-end adoption of AWS, securely maximizing enterprise-wide transformation, innovation, and business value delivery at speed and at scale.

Capabilities and Service Offerings

Help clients manage data as a valuable resource. Grant Thornton uses the AWS Database Migration Service to execute homogeneous (e.g. Oracle to Oracle) and heterogeneous (e.g. MS SQL to AWS Aurora) database cloud migrations. This service includes delivery of cloud-based data management services, including data governance, data architecture, data modeling and design, data security, master data management, data integration, data ETL and ELT, data warehousing, data lakes, enterprise business intelligence, data quality and other services. Grant Thornton delivers this service using AWS database, storage, migration and transfer, content delivery, analytics, machine learning and related services.

Help clients assess their cloud foundation capabilities and maturity and guide client infrastructure and operations (I&O) technical professionals (who are responsible for enterprise cloud migration to IaaS and PaaS) through requirements that enterprises should address in order to support the demands of production workloads on public cloud providers. Grant Thornton help clients understand business and I&O team responsibilities and assess and plan necessary improvements in 12 foundational areas: (1) master and sub-account hierarchies and billing (2) user identity and role-based access control (3) core IP transit and name resolution (4) object, block and file storage (5) application-specific networking (6) compute (7) PaaS services (8) IT operations management (9) continuity, availability, and disaster recovery (10) governance (11) security (12) automation

Help clients establish or mature a cloud center of excellence (CCoE) that is a service and practice center with deep and wide knowledge of cloud development, operations, security, platforms, design, delivery and run best practices. Grant Thornton helps clients establish CCoEs that accelerate businesses innovation through provision of automation, templates, practices, solutions and services that support cloud adoption and usage. With the goal of improving outcomes and managing risks, Grant Thornton helps clients establish CCoEs that provide centralized governance, set cloud policy, guide provider selection, assist with solution architecture and workload placement, and raise the level of cloud knowledge and capture and disseminate best practices. CCoEs established through this service, work in a consultative capacity -- with central IT, business-unit IT, and cloud service consumers -- to help drive cloud-enabled IT transformation.

Help clients understand how cloud adoption transforms the way they work, identify stakeholder barriers and gaps to cloud adoption, recognize cross-organization cloud adoption dependencies, and develop a customize a execution action plan for cloud adoption. Grant Thornton delivers this service to help clients identify and classify cloud adoption challenges from six perspectives (business, people, governance, platform, security, operations) and define and prioritize actions that need to be taken to overcome these cloud adoption challenges. The service based on AWS Cloud Adoption Framework (CAF).
